开发文档

最新版本:

natpass主要用于居家办公，远程开发等使用场景，目前已支持以下功能：

  1. web shell: 通过shell的方式控制远程机器
  2. web vnc: 对于桌面类的操作系统，可实现远程桌面的控制
  3. code-server: 可通过web的方式来访问远端机器上已安装好的code-server服务
